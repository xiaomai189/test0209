# test0209

这是一个 Node.js 学习项目，用于熟悉 GitHub 的使用和 Node.js 基础开发。

## 项目结构

```
test0209/
├── src/
│   ├── test.js    # Node.js 示例程序
│   └── test.py    # Python 示例程序
└── README.md      # 项目说明文档
```

## 快速开始

### 运行 Node.js 程序

```bash
node src/test.js
```

### 运行 Python 程序

```bash
python src/test.py
```

## Node.js 程序说明

`src/test.js` 包含以下示例：

1. **Hello World** - 基础输出
2. **文件系统操作** - 使用 `fs` 模块读取文件
3. **ES6+ 语法** - 箭头函数、模板字符串
4. **模块导出** - `module.exports` 使用
